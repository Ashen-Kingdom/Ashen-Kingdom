# Ashen Kingdom Lobby

A mobile-friendly 3D kingdom lobby built with Three.js.

## Files
- `index.html`
- `style.css`
- `main.js`
- `README.md`

## Run
Open `index.html` in a browser, or deploy the repository to GitHub Pages.

## Controls
- Drag to rotate
- Pinch or wheel to zoom

## Notes
This is a browser prototype for a fantasy strategy lobby scene.
